Навигация по файлам:
1) baseline.ipynb -- бейзлайн-решение задачи;
2) preprocessing.ipynb -- предобработка данных
и добавление новых признаков;
3) main_model.ipynb -- наилучшая модель;
4) sub_model.ipynb -- другая тестовая модель;
5) submission1.csv -- предсказания наилучшей модели;
6) submission2.csv -- предсказания тестовой модели;
7) sample_solution.csv -- бейзлайн-предсказание;
8) test_dataset_test.csv -- данные для тестирования;
9) test.csv -- предобработанные данные для тестирования.
